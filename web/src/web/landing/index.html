2nd COPO Workshop Agenda - 12th/13th December 2016

Day 1

Short project and group introduction
  • Set out what we want to achieve
  • Experimental metadata collection (spreadsheets, ISA standard)
  • Spreadsheet annotation technologies (hypothes.is, RightField, OntoMaton)
  • Raw sequence data submissions (ENA)
Group Breakout
  • Real world sequence datasets that the groups are able to/want to share
  • Understand breadth of experimental data collection and opportunities to standardise

Day 2

Continue group breakouts
  • Start working on annotating experimental metadata using hypothes.is/annotator.js
  • Use Ontology Lookup Service (OLS) to search and use ontology terms for annotation
  • Use Zooma (https://www.ebi.ac.uk/spot/zooma/) to collate terms that donâ€™t exist in ontologies
  • Use COPO to submit ENA data

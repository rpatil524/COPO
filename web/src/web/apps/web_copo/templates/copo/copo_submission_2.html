Submissions - Getting started

The submission component of a profile displays submissions created (conclusive or pending) within a profile.

To create a submission entry, you have to first go through the process of datafile description (under the datafile component). A submission entry is created when the user completes the task of metadata attribution (within the description wizard), and selects to initiate submission of the described datafiles.
